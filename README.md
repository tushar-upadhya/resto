# Resto

This is a simple web application built with ReactJS and Vite. It showcases the use of dummy data to create a responsive restaurant website.

## Features

- Responsive design that adapts to different screen sizes
- Use of ReactJS and Vite for fast and efficient development
- Integration of Tailwind CSS for easy styling and customization



## Dummy Data

The web app uses dummy data to simulate restaurant information such as  recipe, price, and contact details. 

## Technologies Used

- ReactJS
- Vite
- Tailwind CSS
